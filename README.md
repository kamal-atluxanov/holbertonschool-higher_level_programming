# Holberton School - Higher Level Programming

This repository contains projects and exercises for **Higher Level Programming** at Holberton School. 
It covers **SQL**, **Python programming**, **data structures**, **object-oriented programming**, **networking**, **serialization**, **server-side rendering**, and **RESTful APIs**.


##  SQL Projects

### 1. **SQL_introduction**
**Description:**
Covers the basics of SQL, including creating tables, inserting data, and simple SELECT queries.
**Goal:**
Learn to interact with databases and retrieve information efficiently.

### 2. **SQL_more_queries**
**Description:**
Covers advanced SQL queries, joins, group by, aggregate functions, and subqueries.
**Goal:**
Gain deeper understanding of querying databases for complex data extraction.


##  Python Basics

### 3. **python-hello_world**
**Description:**
Introduces Python syntax and writing your first program.
**Goal:**
Get familiar with Python programming environment and basic syntax.

### 4. **python-if_else_loops_functions**
**Description:**
Covers conditional statements, loops, and defining functions.
**Goal:**
Learn to control program flow and create reusable code blocks.

### 5. **python-import_modules**
**Description:**
Covers importing and using Python modules and standard libraries.
**Goal:**
Understand how to extend Python programs with existing libraries.

### 6. **python-exceptions**
**Description:**
Introduces exception handling in Python, using try/except blocks.
**Goal:**
Handle errors gracefully and prevent program crashes.


##  Python Data Structures & OOP

### 7. **python-data_structures**
**Description:**
Covers lists, tuples, sets, and dictionaries in Python.
**Goal:**
Learn to store, access, and manipulate data efficiently.

### 8. **python-more_data_structures**
**Description:**
Covers nested and complex data structures, including lists of dictionaries.
**Goal:**
Work with complex real-world data effectively.

### 9. **python-more_classes**
**Description:**
Introduces more advanced class concepts, class methods, and instance methods.
**Goal:**
Learn object-oriented programming techniques in Python.

### 10. **python-inheritance**
**Description:**
Covers inheritance, subclassing, and extending classes.
**Goal:**
Understand code reuse and polymorphism.

### 11. **python-more_classes**
**Description:**
Covers advanced OOP topics like multiple classes interaction, private attributes, and encapsulation.
**Goal:**
Build well-structured and maintainable object-oriented programs.


##  Python Input/Output & Serialization

### 12. **python-input_output**
**Description:**
Covers reading from and writing to files in Python.
**Goal:**
Learn to handle data persistence using files.

### 13. **python-serialization**
**Description:**
Covers converting Python objects to and from serial formats (JSON, pickle).
**Goal:**
Enable storing or transmitting Python objects between programs or systems.


##  Python Networking & Web

### 14. **python-network_1**
**Description:**
Covers basic networking concepts and socket programming in Python.
**Goal:**
Understand how Python programs communicate over networks.

### 15. **python-server_side_rendering**
**Description:**
Covers generating dynamic web content using Python on the server side.
**Goal:**
Learn to serve HTML pages dynamically from Python applications.

### 16. **restful-api**
**Description:**
Introduces building RESTful APIs using Python frameworks.
**Goal:**
Learn to create APIs that allow other programs to interact with your application over HTTP.


**Tip:** Follow this order to build your skills step by step:
**SQL → Python Basics → Data Structures → OOP → File I/O & Serialization → Networking → Server-side Rendering → RESTful APIs**
